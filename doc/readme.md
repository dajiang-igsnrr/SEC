# Code Documentation for Fortran Project
This folder contains auto-generated documentation (HTML) for Fortran source code, created using **Doxygen**. The documentation includes detailed code annotations, structural breakdowns, and function call relationships.

## How to View the Documentation
### 1. HTML Documentation (Interactive Version)
1. Download the entire documentation package to your **local computer**.
2. Locate the `html` folder in the downloaded directory.
3. Double-click the `index.html` file, or right-click it and select "Open with" to choose any modern web browser (Chrome, Firefox, Edge, Safari, etc.).
4. You can now browse:
   - Full Fortran code structure (modules, subroutines, functions)
   - Detailed comments for each code element
   - Parameter descriptions (input/output, data types, purpose)
   - Visual function call graphs and dependencies
   - Searchable content (via the built-in search bar in HTML)
